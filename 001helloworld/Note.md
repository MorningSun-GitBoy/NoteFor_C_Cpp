# 001halloworld
```
 C语言的程序结构分为两部分，即库引入和函数调用。C++再此基础上增加了命名空间的指定。现在出现了三个新的名词：库，函数和命名空间。
      库：已经定义好的函数集（C++中还包括类与对象集）
      函数：按部就班的一系列步骤
      命名空间：已经确定了代表意义的字符串集，以及和它们共用的可不重复命名的空间
 我的解释并不完善，甚至还引入了更多的新名词。但是你只要记住，库是已经写好的东西，而命名空间决定C++文件中的命名就足够了。
 对于C和C++中实现“Hallo World!”的输出，实际上采用了不同的方式（C调用printf函数，而C++使用cout对象与"字符串"的">>"运算完成）。但是作为初学者，只需掌握共同点，即都是用库中已经写好的代码实现输入输出，我们以后的输入输出也都会依赖这些库。
 我们之后还会学到类、对象等C++中的这些数据结构的知识，它们会带给大家全新的认识。
```
